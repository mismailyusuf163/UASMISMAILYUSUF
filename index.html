Project Saya

Shift Cassar

Shift Cassar merupakan website yang saya kembangkan sebagai media digital untuk membantu membuat enksrpsi berbasis shift caesar. Website ini dibuat dengan mengutamakan tampilan yang sederhana, ,Serta mempelajari dasar dari enigma machine dan hanya sebagai contoh di perangkat mobile. Selama proses pengembangannya saya menerapkan konsep antarmuka yang bersih (Clean UI), navigasi yang mudah dipahami, serta optimasi performa agar website dapat berjalan dengan cepat. Project ini menjadi salah satu bentuk pengembangan kemampuan saya dalam membangun website menggunakan HTML, CSS, dan phyton.
